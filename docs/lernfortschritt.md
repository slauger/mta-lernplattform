# Wie ich meinen Lernfortschritt dokumentiere

## Warum Fortschritt dokumentieren?

Das Dokumentieren deines Lernfortschritts hilft dir:

- ✅ **Motivation**: Du siehst, was du bereits geschafft hast!
- ✅ **Struktur**: Du behältst den Überblick
- ✅ **Lücken erkennen**: Du siehst, wo du noch nacharbeiten musst
- ✅ **Prüfungsvorbereitung**: Du weißt, was du wiederholen musst

!!! success "Tipp"
    Nimm dir **jeden Sonntag 15 Minuten** Zeit, um deinen Fortschritt zu aktualisieren!

---

## Methode 1: Checklisten (einfach)

### Gesamt-Checkliste

Nutze diese Checkliste, um deinen Fortschritt durch alle Kapitel zu tracken:

#### Einführung
- [ ] Einführung & Orientierung gelesen
- [ ] Begriffe & Abkürzungen durchgearbeitet (mind. 50 Begriffe gelernt)

#### Theorie
- [ ] Ziele, Aufgaben & Werte verstanden
- [ ] Brennen & Löschen verstanden (Brandklassen, Löschmittel)
- [ ] Fahrzeuge & Geräte gekannt (LF, HLF, PSA)
- [ ] Rechtsgrundlagen & Sicherheit gelernt (BayFwG, UVV, GAMS)
- [ ] Einsatz & Funk verstanden (Einsatzablauf, Funksprüche)

#### Praxis
- [ ] Mastwurf sicher unter 10 Sekunden
- [ ] Zimmermannsschlag sicher unter 30 Sekunden
- [ ] Achterknoten (Endacht + Bucht-Achter) beherrscht
- [ ] Webleinstek ohne Verwechslung mit Mastwurf
- [ ] Fitness-Training begonnen (mind. 2x/Woche)
- [ ] Materialliste abgearbeitet (Übungsleine, Lehrbuch)

#### Prüfungsvorbereitung
- [ ] Mind. 50 Prüfungsfragen beantwortet
- [ ] Mind. 100 Prüfungsfragen beantwortet
- [ ] Alle 120+ Prüfungsfragen beantwortet
- [ ] Falsche Fragen wiederholt

#### Goldkronach
- [ ] Goldkronach-Kapitel gelesen
- [ ] Mit Kommandant über lokale Fahrzeuge gesprochen
- [ ] Gerätehaus-Rundgang gemacht

#### Ressourcen
- [ ] Lehrbuch gekauft/ausgeliehen
- [ ] Feuerwehr-Lernbar ausprobiert
- [ ] YouTube-Kanäle abonniert
- [ ] FwDV 3 gelesen

### Kapitel-Tracker

Für jedes Theorie-Kapitel:

**Beispiel: Brennen & Löschen**

| Thema | Gelesen | Verstanden | Quiz bestanden | Wiederholung nötig? |
|-------|---------|-----------|----------------|---------------------|
| Verbrennungsdreieck | ✅ | ✅ | ✅ | Nein |
| Brandklassen A-F | ✅ | ✅ | ❌ (nur 60%) | Ja, Klasse D + F wiederholen |
| Löschmittel | ✅ | ⚠️ (teilweise) | ❌ | Ja, Schaum-Wirkung unklar |
| Brandphasen | ✅ | ✅ | ✅ | Nein |
| Backdraft | ✅ | ⚠️ | ❌ | Ja, Anzeichen lernen |

**Legende:**
- ✅ = Ja / Gut
- ⚠️ = Teilweise / Okay
- ❌ = Nein / Schlecht

## Methode 3: Git/GitHub (für Technik-Affine)

Wenn du mit Git vertraut bist, kannst du deinen Lernfortschritt in diesem Repo tracken!

### Schritt 1: Repo initialisieren

```bash
cd /Users/simon/git/feuerwehr
git init
git add .
git commit -m "Initial commit: MTA Lernmaterial"
```

### Schritt 2: Lernfortschritt-Datei anlegen

Erstelle eine Datei `LERNFORTSCHRITT.md`:

```markdown
# Mein Lernfortschritt

## Woche 1 (01.01.-07.01.2026)
- ✅ Einführung gelesen
- ✅ Mastwurf 20x geübt
- ⏳ Brandklassen lernen (in Arbeit)

## Woche 2 (08.01.-14.01.2026)
- ✅ Brandklassen A-F gelernt
- ✅ Zimmermannsschlag 15x geübt
- ❌ Fitness: Nur 1x Joggen (Ziel: 3x) → nächste Woche besser!
```

### Schritt 3: Regelmäßig committen

```bash
git add LERNFORTSCHRITT.md
git commit -m "Woche 2: Brandklassen gelernt, Knoten geübt"
```

**Vorteil:** Du hast eine vollständige History deines Lernfortschritts!

## Methode 4: Digitale Tools

### Notion / Obsidian

- Erstelle eine Datenbank mit allen Themen
- Markiere Status (Todo / In Progress / Done)
- Verlinke Notizen untereinander

### Trello / Kanban-Board

- **Todo**: Themen, die du noch lernen musst
- **In Progress**: Was du gerade lernst
- **Done**: Was du abgeschlossen hast

**Beispiel-Board:**
```
| Todo             | In Progress      | Done                  |
|------------------|------------------|-----------------------|
| - Fahrzeugkunde  | - Brandklassen   | - Mastwurf            |
| - Funksprüche    | - Fitness 3x/Wo  | - Einführung gelesen  |
| - Prüfungsfragen |                  | - Begriffe (50 Stück) |
```

### Spreadsheet (Excel / Google Sheets)

Erstelle eine Tabelle wie oben (Wochen-Tracker), aber digital → einfacher zu aktualisieren!

## Methode 6: Lerntagebuch

Führe ein Lerntagebuch (handschriftlich oder digital):

**Beispiel-Eintrag:**

```
Datum: 10.01.2026
Dauer: 2 Stunden

Was habe ich heute gelernt?
- Brandklassen A-F
- Unterschied zwischen Retten und Bergen
- Mastwurf 10x geübt

Was lief gut?
- Brandklasse A+B sofort verstanden
- Mastwurf unter 10 Sekunden!

Was lief schlecht?
- Brandklasse D (Metalle) noch unklar
- Zimmermannsschlag dauert noch zu lang (40 Sek.)

Was mache ich morgen?
- Brandklasse D nochmal lesen
- Zimmermannsschlag 20x üben
- 30 min Joggen
```

## Mini-Checklisten pro Kapitel

### Kapitel: Brennen & Löschen

- [ ] Verbrennungsdreieck verstanden
- [ ] Alle 5 Brandklassen (A, B, C, D, F) gelernt
- [ ] Löschmittel für jede Brandklasse kennen
- [ ] Weiß, warum NIEMALS Wasser auf Fett
- [ ] Weiß, warum NIEMALS Wasser auf Metalle
- [ ] 4 Löschwirkungen (Kühlen, Ersticken, Verdünnen, Antikatalytisch) kennen
- [ ] Backdraft-Anzeichen kennen
- [ ] Quiz zu diesem Kapitel bestanden (mind. 80%)

### Kapitel: Fahrzeuge & Geräte

- [ ] Unterschied TSF, LF, TLF, HLF kennen
- [ ] Weiß, dass LF 20 für 9 Personen ist
- [ ] B-Schlauch (75 mm) vs. C-Schlauch (42/52 mm) kennen
- [ ] PSA-Bestandteile (6 Teile) aufzählen können
- [ ] Weiß, wie lange ein PA hält (20-30 min)
- [ ] Unterschied Saugschlauch/Druckschlauch kennen
- [ ] Quiz zu diesem Kapitel bestanden (mind. 80%)

(... für alle Kapitel)

## Tipps zum Durchhalten

### 1. Realistische Ziele setzen

**Schlecht:** "Ich lerne diese Woche alles!"

**Gut:** "Diese Woche lerne ich Brandklassen + übe Mastwurf 3x"

### 2. Kleine Schritte

**Besser 15 Minuten täglich** als 3 Stunden am Sonntag!

### 3. Belohnungen

Nach jeder abgeschlossenen Woche: Lieblingsessen, Film, etc.

### 4. Lernpartner

Finde einen MTA-Kollegen und lernt zusammen! Ihr könnt euch gegenseitig abfragen.

### 5. Routine

Feste Lernzeiten (z. B. täglich 19:00-19:30 Uhr) helfen, am Ball zu bleiben.
