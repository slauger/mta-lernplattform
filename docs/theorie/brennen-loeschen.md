## Was ist Verbrennung?

**Definition:** Verbrennung ist eine **chemische Reaktion** zwischen einem brennbaren Stoff und Sauerstoff unter Freisetzung von Wärme und Licht.

### Das Verbrennungsdreieck

Damit ein Feuer brennen kann, braucht es **drei Voraussetzungen**:

```
        Brennbarer Stoff
              / \
             /   \
            /     \
           /       \
          /         \
         /   FEUER   \
        /_____________\
   Sauerstoff    Zündtemperatur
```

1. **Brennbarer Stoff** (z. B. Holz, Benzin, Gas)
2. **Sauerstoff** (mind. 15% in der Luft)
3. **Zündtemperatur** (ausreichende Wärme zum Entzünden)

!!! tip "Merkregel"
    Fehlt **eine** der drei Komponenten, kann kein Feuer brennen!

### Das Verbrennungstetraeder (erweitert)

Moderne Brandlehre fügt eine vierte Komponente hinzu:

```
        Brennbarer Stoff
              /|\
             / | \
            /  |  \
           / Ketten- \
          / reaktion  \
         /      |      \
        /_______|_______\
   Sauerstoff    Zündtemperatur
```

**4. Kettenreaktion:** Die chemische Reaktion hält sich selbst aufrecht (Radikale).

!!! info "Warum wichtig?"
    Wenn du eine der vier Komponenten entfernst, erlischt das Feuer. Das ist die Grundlage des Löschens!

## Brandklassen

Brennbare Stoffe werden nach ihrem **Aggregatzustand** in fünf Brandklassen eingeteilt. Jede Brandklasse erfordert **spezielle Löschmittel**!

| Brandklasse | Stoffart | Beispiele | Löschmittel |
|-------------|----------|-----------|-------------|
| **A** | Feste, glutbildende Stoffe | Holz, Papier, Textilien, Stroh | Wasser, Schaum, Pulver |
| **B** | Flüssige/flüssig werdende Stoffe | Benzin, Öl, Wachs, Teer | Schaum, Pulver, CO₂ |
| **C** | Gasförmige Stoffe | Erdgas, Propan, Wasserstoff | Gaszufuhr absperren, Pulver |
| **D** | Brennbare Metalle | Aluminium, Magnesium, Natrium | Metallbrandpulver, Sand |
| **F** | Speiseöle und -fette | Friteuse, Pfanne | Fettbrandlöscher, Deckel |

### Brandklasse A – Feste Stoffe

**Merkmale:**
- Brennen mit Flammen **und** Glut
- Hinterlassen Asche
- Wasser ist sehr effektiv

**Beispiele:**
- Holz
- Papier
- Textilien (Kleidung, Vorhänge)
- Stroh, Heu
- Kohle

**Löschmittel:** Wasser, Schaum, ABC-Pulver

!!! success "Wasser ist ideal"
    Bei Brandklasse A ist Wasser das beste Löschmittel (kühlend, kostengünstig, überall verfügbar).

---

### Brandklasse B – Flüssigkeiten

**Merkmale:**
- Brennen nur mit Flammen (keine Glut)
- Oft hohe Brandtemperaturen
- Breiten sich schnell aus

**Beispiele:**
- Benzin, Diesel
- Heizöl, Motoröl
- Lacke, Farben
- Alkohol
- Wachs (schmilzt und wird flüssig)

**Löschmittel:** Schaum, ABC-Pulver, BC-Pulver, CO₂

!!! danger "NIEMALS Wasser auf brennende Flüssigkeiten!"
    Wasser ist schwerer als Benzin → Benzin schwimmt oben → Feuer breitet sich explosionsartig aus!

**Richtig löschen:**
- **Schaum** legt sich auf die Flüssigkeit und erstickt das Feuer
- **Pulver** unterbricht die Kettenreaktion

---

### Brandklasse C – Gase

**Merkmale:**
- Brennen nur mit Flammen
- Oft unter Druck (Explosion sgefahr!)
- Können unsichtbar sein

**Beispiele:**
- Erdgas (Methan)
- Propan (Gasflasche)
- Butan (Campingkocher)
- Wasserstoff
- Acetylen

**Löschmittel:** Gaszufuhr absperren, ggf. Pulver

!!! warning "Erst absperren, dann löschen!"
    Wenn du ein brennendes Gas löschst, **ohne die Zufuhr abzusperren**, strömt weiter Gas aus → **Explosionsgefahr**!

**Richtiges Vorgehen:**
1. Gaszufuhr absperren (Ventil schließen)
2. Dann ggf. löschen (meist erlischt das Feuer von selbst)

---

### Brandklasse D – Metalle

**Merkmale:**
- Sehr hohe Temperaturen (bis 3000°C!)
- Reagieren heftig mit Wasser → **Explosion!**
- Selten, aber sehr gefährlich

**Beispiele:**
- Aluminiumpulver
- Magnesiumpulver
- Natrium, Kalium (reagieren mit Wasser!)

**Löschmittel:** Metallbrandpulver (z. B. M 28), trockener Sand, Zement

!!! danger "NIEMALS Wasser auf Metallbrände!"
    Wasser reagiert mit brennenden Metallen → Knallgasexplosion!

---

### Brandklasse F – Speiseöle und -fette

**Merkmale:**
- Sehr hohe Temperaturen (über 300°C)
- Fettexplosion bei Kontakt mit Wasser!
- Häufig in Küchen (Friteuse, Pfanne)

**Beispiele:**
- Friteusenöl
- Bratfett in der Pfanne
- Speiseöl

**Löschmittel:** Fettbrandlöscher, Deckel auf die Pfanne

!!! danger "NIEMALS Wasser auf Fettbrände!"
    Wasser verdampft schlagartig (1 Liter Wasser → 1700 Liter Dampf) → **Fettexplosion** (Stichflamme bis zur Decke!)

**Richtiges Vorgehen:**
1. **Herd ausschalten**
2. **Deckel auf die Pfanne** (erstickt das Feuer)
3. **Fettbrandlöscher** verwenden (bildet eine Seifenschicht)
4. **NIEMALS Wasser** verwenden!

---

## Löschmittel

### 1. Wasser

**Eigenschaften:**
- Überall verfügbar
- Kostengünstig
- Sehr effektiv bei Brandklasse A

**Löschwirkung:** **Kühlen** (Hauptwirkung) + Ersticken (Nebelstrahl)

**Geeignet für:** Brandklasse A

**Nicht geeignet für:** B, C, D, F, elektrische Anlagen (Stromunfälle!)

**Einsatzformen:**
- **Vollstrahl**: große Reichweite, kühlt stark
- **Sprühstrahl**: große Oberfläche, kühlt und erstickt
- **Nebelstrahl**: sehr fein zerstäubt, kühlt und verdrängt Rauch

---

### 2. Schaum

**Eigenschaften:**
- Besteht aus Wasser + Schaummittel + Luft
- Schwimmt auf Flüssigkeiten
- Bildet erstickende Schicht

**Löschwirkung:** **Ersticken** (Hauptwirkung) + Kühlen

**Geeignet für:** Brandklasse A und B

**Schaumarten:**
- **Schwerschaum**: dichter, stabil, für Flächenbrände
- **Mittelschaum**: universell einsetzbar
- **Leichtschaum**: große Volumina, für Raumflutung

!!! info "Schaum bei Benzinbränden"
    Schaum legt sich auf das brennende Benzin und erstickt das Feuer – perfekt für Klasse B!

---

### 3. Pulver

**Eigenschaften:**
- Chemisches Pulver (verschiedene Typen)
- Sehr effektiv, aber verschmutzt stark
- Wirkt schlagartig

**Löschwirkung:** **Antikatalytische Wirkung** (unterbricht Kettenreaktion) + Ersticken

**Typen:**
- **ABC-Pulver**: für Brandklassen A, B, C
- **BC-Pulver**: für Brandklassen B, C
- **D-Pulver (Metallbrandpulver)**: nur für Brandklasse D

**Geeignet für:** A, B, C (je nach Typ), D (Spezialpulver)

!!! warning "Nachteil: Verschmutzung"
    Pulver verdreckt alles und hinterlässt korrosive Rückstände. Nur einsetzen, wenn nötig!

---

### 4. Kohlendioxid (CO₂)

**Eigenschaften:**
- Farb- und geruchloses Gas
- Verdrängt Sauerstoff
- Hinterlässt keine Rückstände

**Löschwirkung:** **Ersticken** (verdrängt Sauerstoff)

**Geeignet für:** Brandklasse B (nur kleine Brände), elektrische Anlagen

**Nicht geeignet für:** Brandklasse A (Glut brennt weiter!)

!!! danger "Erstickungsgefahr!"
    CO₂ verdrängt Sauerstoff → in geschlossenen Räumen Erstickungsgefahr!

## Die vier Löschwirkungen

Um ein Feuer zu löschen, musst du mindestens eine Komponente des Verbrennungstetraeders entfernen. Das erreichst du durch vier Löschwirkungen:

### 1. Kühlen

**Prinzip:** Temperatur unter die Zündtemperatur senken.

**Wie?** Wärme entziehen (z. B. mit Wasser).

**Beispiel:** Wasser kühlt brennendes Holz → Temperatur sinkt → Feuer erlischt.

**Löschmittel:** Wasser, Schaum

---

### 2. Ersticken

**Prinzip:** Sauerstoffzufuhr unterbinden.

**Wie?** Luftzufuhr abdecken oder Sauerstoff verdrängen.

**Beispiel:** Deckel auf brennende Pfanne → kein Sauerstoff → Feuer erlischt.

**Löschmittel:** Schaum, CO₂, Löschdecke, Pulver

---

### 3. Verdünnen (selten)

**Prinzip:** Brennbaren Stoff mit nicht brennbarem Stoff vermischen.

**Wie?** Z. B. brennbare Gase mit Luft/Inertgas verdünnen.

**Beispiel:** Bei Gaslecks wird manchmal mit Wasser „verdünnt" (Gase werden weggespült).

**Hinweis:** In der Praxis selten angewendet!

---

### 4. Antikatalytische Wirkung (chemische Reaktion hemmen)

**Prinzip:** Kettenreaktion der Verbrennung unterbrechen.

**Wie?** Chemische Stoffe (z. B. Pulver) reagieren mit den Radikalen und stoppen die Reaktion.

**Beispiel:** Pulver unterbricht die Kettenreaktion → Feuer erlischt schlagartig.

**Löschmittel:** Pulver

## Brandphasen

Ein Brand durchläuft typischerweise **drei Phasen**:

### 1. Entstehungsbrand (Initialphase)

- Feuer beginnt klein (z. B. brennende Kerze)
- Temperatur steigt langsam
- Rauchentwicklung gering
- **Löschen einfach** (z. B. mit Feuerlöscher)

**Beispiel:** Adventskranz fängt Feuer → mit Wasserglas löschen.

---

### 2. Vollbrand

- Feuer hat sich ausgebreitet
- Alle brennbaren Stoffe im Raum brennen
- Temperatur bis 1000°C
- **Löschen schwierig**, Einsatz der Feuerwehr nötig

**Beispiel:** Wohnzimmer steht komplett in Flammen.

---

### 3. Brandende (Abkühlphase)

- Brennbare Stoffe sind verbraucht
- Temperatur sinkt
- Nur noch Glut vorhanden
- **Rückzündungsgefahr!** (Glut kann wieder aufflammen)

**Beispiel:** Nach dem Löschen glimmt noch Glut in den Möbeln → muss abgelöscht werden.

---

## Gefahren an der Brandstelle

### 1. Atemgifte

**Gefahr:** Giftiger Rauch (Kohlenmonoxid, Blausäure, Chlorgas)

**Wirkung:** Bewusstlosigkeit, Tod (schon nach wenigen Atemzügen!)

**Schutz:** Atemschutzgerät (PA)

!!! danger "Rauch tötet schneller als Feuer!"
    Die meisten Brandopfer sterben durch **Rauch**, nicht durch Flammen!

---

### 2. Wärme/Hitze

**Gefahr:** Temperaturen bis 1000°C, Verbrennungen, Hitzeschlag

**Wirkung:** Verbrennungen, Kreislaufzusammenbruch

**Schutz:** PSA (Schutzkleidung), kurze Einsatzzeiten, Abkühlung

---

### 3. Fehlender Sauerstoff

**Gefahr:** Sauerstoffmangel in Brandräumen (unter 15%)

**Wirkung:** Bewusstlosigkeit, Tod

**Schutz:** Atemschutzgerät

---

### 4. Einsturzgefahr

**Gefahr:** Schwächung der Gebäudestruktur durch Feuer

**Wirkung:** Einsturz von Decken, Wänden, Dächern

**Schutz:** Vorsicht, Rückzug bei Gefahr, Statiker hinzuziehen

---

### 5. Rauchgasdurchzündung

**Gefahr:** Heiße Rauchgase entzünden sich plötzlich (Flashover).

**Auslöser:** Raum erhitzt sich → Rauchgase werden so heiß (ca. 600°C), dass sie sich selbst entzünden.

**Wirkung:** Schlagartige Feuerausbreitung im gesamten Raum.

**Schutz:** Tür **langsam** öffnen, Temperatur prüfen (Handrücken an Tür), ggf. sofortiger Rückzug.

!!! danger "Flashover"
    Innerhalb von Sekunden steht der gesamte Raum in Flammen!

---

### 6. Backdraft (Rauchgasexplosion)

**Gefahr:** Explosion durch plötzliche Sauerstoffzufuhr.

**Auslöser:** Feuer brennt in geschlossenem Raum → Sauerstoff wird verbraucht → Feuer schwelt → heiße, unverbrannte Gase sammeln sich → Tür wird geöffnet → Sauerstoff strömt ein → **EXPLOSION**!

**Anzeichen:**
- Schwarzer, öliger Rauch
- Rauch „pulsiert" (wird rein- und rausgesaugt)
- Fenster verrußt von innen
- Kaum Flammen sichtbar

**Schutz:** Tür NICHT öffnen, Wand durchbrechen und Wasser von außen einbringen.

!!! danger "Lebensgefahr!"
    Backdraft kann ganze Trupps töten. **Immer auf Anzeichen achten!**

## Zusammenfassung

| Thema | Kernaussage |
|-------|-------------|
| **Verbrennungsdreieck** | Brennbarer Stoff + Sauerstoff + Zündtemperatur |
| **Brandklassen** | A (fest), B (flüssig), C (gasförmig), D (Metall), F (Fett) |
| **Löschmittel** | Wasser (A), Schaum (A/B), Pulver (A/B/C/D), CO₂ (B) |
| **Löschwirkungen** | Kühlen, Ersticken, Verdünnen, Antikatalytisch |
| **Größte Gefahr** | Rauch (Atemgifte!) |
| **Backdraft** | Rauchgasexplosion durch plötzliche Sauerstoffzufuhr |

??? question "2. Welche Brandklasse hat Benzin?"
